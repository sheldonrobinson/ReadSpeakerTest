Metahuman Lip Sync

Note, this is an experimental feature and is currently only available for UE4.27. To install it, go to the download page and download the required module found under 'Modules->MetaHumanLipSync'.

This section explains the ReadSpeaker TTS/Unreal Metahuman lip-sync integration. Begin by installing Metahuman to your Unreal project. Once installed, you can proceed to install ReadSpeakerTTS and follow the guidelines for installing voice engines into the project. Download the 'MetaHumanLipSync' model, extract and then move/copy the contents of MetaHumanLipSync to {PLUIN_ROOT}/Content/. The next step to get the MetaHuman lip-synced with the speech audio produced by ReadSpeakerTTS is to attach a TTSSpeaker component to the Metahuman actor.

Make sure to configure the speech characteristics in the "Characteristics" tab in the Details panel. Next, go to your Metahuman's 'Face' component by selecting the Metahuman and go going to 'Root->Body->Face' in the Details panel. Under the Animation tab, select 'Use Animation Blueprint' for the 'Animation Mode'. Set the 'Anim Class' to 'RS_Face_AnimBP' you may utilize the search bar in order to find it. The result should look like this:

Now the TTSSpeaker component attached to the Metahuman actor will be linked with the Animation Blueprint attached to the Metahuman face. Any speech you produce using the TTSSpeaker component will automatically lip-sync the Metahuman with the speech. You may try it out by editing the Metahuman actor blueprint and set up a call to the 'Say' or 'SayAsync' function.
